> Hybrid RAG system combining vector search and graph reasoning for complex legal queries

- Agentic workflow with **parallel retrieval, caching, and failure-aware orchestration** using LangGraph
- Achieved **93% accuracy** on complex queries and **98%** on simple queries
- Reduced end-to-end latency by **7 seconds** via hot-path pruning and parallel retrieval
- **Stack:** Python · Flask · LangGraph · OpenAI GPT-OSS-120B · ChromaDB · Neo4j AuraDB · Redis

- Autonomous agents for resume parsing, JD structuring, XAI-based shortlisting, and scheduling
- Human-in-the-loop validation via **LangGraph workflows** with feedback-driven optimization

- **Stack:** Python · Flask · LangGraph · React · SQLAlchemy · ChromaDB · PyTorch · MongoDB
